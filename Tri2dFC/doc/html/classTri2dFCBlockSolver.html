Tri2dFCBlockSolver Class Reference

Class Tri2dFCBlockSolver holds the data and specifies the operations that will be carried out on a partitioned block of triangular mesh. When initialized it allocates data and creates communicators between data that needs to be exchanged between blocks. The other operations define things that need to be done to compute a solution on each block. More...
